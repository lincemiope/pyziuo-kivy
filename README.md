# README #

UO.dll wrapper + Kivy based interface

### What is this repository for? ###

PyziUO is a python UO assistant that uses UO.dll.
Version: alpha - kivy 1.10.0 - python 3.6.1


### How do I get set up? ###

You'll need:
- python 3.6.1
- kivy 1.10.0 and its dependencies
- pyperclip
